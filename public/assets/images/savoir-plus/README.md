# Images page « Savoir plus »

Placez dans ce dossier les 6 images pour la page `/savoir-plus`, avec les noms suivants.

**Si vos images sont ailleurs** (ex. `assets/` du projet Cursor ou un dossier `images/`), copiez-les ici et renommez-les comme ci-dessous.

## Mapping (description → nom du fichier)

| Nom du fichier à utiliser | Contenu de l’image | Utilisation sur la page |
|---------------------------|-------------------|--------------------------|
| `huile-argan.png` | Bouteille ambre, liquide doré, emblème or (C / motif marocain) | Bloc **Huile d'argan** |
| `huile-olive.png` | Bouteille verte, huile dorée, bouchon verseur, plan bois | Bloc **Huile d'olive** |
| `miel-pot-rayon.png` | Pot de miel, cuillère en bois, morceaux de rayon | Bloc **Miel** |
| `amlou-cacahuetes.png` | Pot amlou cacahuètes, amandes, cuillère avec amlou | Bloc **Amlou** + section **Idées de recettes** |
| `coffret-produits.png` | Coffret bois ouvert : amlou, Huile Miracle, pots miel/huile, branches d’olivier | Section **Bienfaits nutritionnels** |
| `huile-miracle-soins-cheveux.png` | Huile Miracle, masseur cuir chevelu, derma roller, gant | Section **Bienfaits pour le corps** |
| `recette-tiramisu-amlou.png` | Dessert en verre (couches, cacao, amandes, miel) | Carousel **Idées de recettes** — Tiramisu amlou |
| `recette-tartines-bowls.png` | Yaourt, granola, tartines, fruits (petit-déjeuner) | Carousel — Tartines & bowls |
| `recette-smoothies-finitions.png` | Poisson, légumes rôtis, smoothie, huiles | Carousel — Smoothies & finitions |
| `dashboard-corps-composite.png` | Visuel composite “tableau de bord” (cheveux/barbe/peau/ongles) | Section **Bienfaits — dashboard premium** |

## Extension

Le code attend des fichiers en **`.png`**. Si vos fichiers sont en `.jpg` ou `.webp`, renommez-les en `.png` ou modifiez les chemins dans `src/app/savoir-plus/page.tsx`.
